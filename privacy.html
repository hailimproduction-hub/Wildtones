Politique de Confidentialité

Dernière mise à jour : 13 avril 2026

1. Introduction

Chez Wildtones, nous prenons votre vie privée très au sérieux. Cette politique explique quelles informations nous collectons, comment nous les utilisons et comment nous les protégeons.

2. Informations Collectées

Nous collectons les types d'informations suivants :

  • Informations de compte TikTok : Token d'accès API, nom d'utilisateur, ID de compte
  • Données d'utilisation : Statistiques de publication, performance des vidéos
  • Informations techniques : Logs de connexion, adresse IP, type d'appareil
  • Préférences : Paramètres de publication, types de contenu préférés

3. Utilisation des Données

Nous utilisons vos données pour :

  • Publier automatiquement du contenu sur votre compte TikTok
  • Optimiser les horaires de publication
  • Analyser les performances de vos vidéos
  • Améliorer notre service et nos algorithmes
  • Vous envoyer des notifications importantes

4. Partage des Données

Nous ne vendons jamais vos données personnelles.

Nous partageons vos données uniquement avec :

  • TikTok : Via leur API officielle pour la publication de contenu
  • Prestataires de services : Hébergement cloud, services d'analyse (anonymisés)
  • Obligations légales : Si requis par la loi

5. Sécurité

Nous mettons en œuvre des mesures de sécurité robustes :

  • Chiffrement SSL/TLS pour toutes les communications
  • Stockage sécurisé des tokens d'accès
  • Authentification à deux facteurs pour l'accès admin
  • Audits de sécurité réguliers

6. Vos Droits

Vous disposez des droits suivants concernant vos données :

  • Accès : Demander une copie de vos données
  • Rectification : Corriger des informations inexactes
  • Suppression : Demander la suppression de vos données
  • Portabilité : Transférer vos données vers un autre service
  • Opposition : Vous opposer à certains traitements

Pour exercer ces droits, contactez-nous à privacy@wildtones.app

7. Conservation des Données

Nous conservons vos données aussi longtemps que vous utilisez notre service. Si vous supprimez votre compte, nous effaçons vos données personnelles dans les 30 jours, sauf obligation légale de conservation.

8. Cookies

Nous utilisons des cookies essentiels pour le fonctionnement du service et des cookies analytiques pour améliorer l'expérience utilisateur. Vous pouvez gérer vos préférences dans les paramètres de votre navigateur.

9. Modifications

Nous pouvons mettre à jour cette politique. Nous vous informerons des changements importants par email ou notification.

10. Contact
